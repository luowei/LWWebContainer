## Installation

LWWebContainer is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LWWebContainer'
```

**Carthage**
```ruby
github "luowei/LWWebContainer"
```
